財務・会計 問題バンク

バッチ3：ファイナンス計算（L1–L2／回答方法の定着版・21問）

各論点は 解き方の型（なぜ→手順→罠) → L1（素直) → L2（罠込み) の順。
解説は毎回この手順を同じ順序でなぞります。手順を覚えるための反復が目的です。各問4点。

1. 税引後キャッシュフロー 〔頻度A〕

解き方の型

  • なぜ：投資の値打ちは「現金」で測る。だが税金は利益に課される。減価償却は現金が出ていかないのに利益を減らす＝税金を減らす（節税)。だから現金ベースに直すと、本体の税引後に「減価の節税分」を足し戻すことになる。
  • 手順：①（必要なら）減価償却費を出す　②次のどちらかで計算
    • 与えられたのが減価償却前の税引前CF（現金収支)なら → 税引後CF＝税引前CF×(1−t)＋減価×t
    • 与えられたのが税引前利益（減価控除後)なら → 税引後CF＝利益×(1−t)＋減価
    • ※この2式は同じもの（利益＝CF−減価を代入すると一致する)
  • 罠：問題文のCFが「減価償却前」か「後（利益)」かを必ず読む。ここを取り違えると全部ずれる。

2. 正味現在価値（NPV) 〔頻度A〕

解き方の型

  • なぜ：将来の現金は今の現金より価値が低い（割引)。各期のCFを「今の価値」に直して合計し、初期投資を引く。プラスなら価値を生む投資。
  • 手順：①各期の（税引後)CFを確定　②現在価値に直す（毎期バラバラ→各期に複利現価係数／毎期一定→年金現価係数を一発で掛ける)　③合計−初期投資＝NPV
  • 罠：(1)毎期一定なら年金現価係数（個別に複利現価を足さない)。(2)最終年の残存価額の売却収入は別枠で足す（複利現価で割引)。(3)初期投資は第0期なので割り引かない。(4)NPVは平気でマイナスになる──プラス前提で選ばない。

3. 投資判定（IRR・回収期間・収益性指数) 〔頻度A〕

解き方の型

  • なぜ：NPV以外にも採否の物差しがある。それぞれ何を見ているかと判定基準を押さえる。
  • 基準：
    • IRR＝NPVがゼロになる割引率。IRR ＞ 資本コスト → 採択。永続CFなら IRR＝CF÷初期投資 で即出る。
    • NPV判定：NPV ＞ 0 → 採択。単独案ではIRR判定とNPV判定は必ず一致（IRR＞r ⟺ NPV＞0)。
    • 収益性指数(PI)＝CF現在価値合計÷初期投資。PI ＞ 1 → 採択。
  • 罠：相互排他的な案ではNPVとIRR・PIが食い違うことがある（投資規模が違うため)。そのときは企業価値最大化のNPVを優先。

4. 加重平均資本コスト（WACC) 〔頻度A〕

解き方の型

  • なぜ：企業は負債と自己資本の両方で調達し、それぞれコストが違う。だから調達割合で加重平均する。負債利子は損金になり節税が効くので、負債コストは実質 (1−t) 倍。
  • 手順：①負債と自己資本の時価ウェイトを出す　②WACC＝自己資本ウェイト×自己資本コスト＋負債ウェイト×負債コスト×(1−t)
  • 罠：(1)「負債比率(D/E)」で与えられたらウェイトに変換。D/E＝0.25 は「負債:自己資本＝1:4」なので、ウェイトは0.2と0.8（0.25ではない)。最頻出の罠。(2)負債コストに (1−t) を掛け忘れない。

5. 配当割引モデル 〔頻度A〕

解き方の型

  • なぜ：株式の価値＝将来もらえる配当の現在価値の合計。
  • 2つの型：
    • 定額（成長なし)：P＝D÷r（同額が永久に続く＝無限等比級数の和)
    • 定率成長（ゴードン)：P₀＝D₁÷(r−g)　D₁は「来期(1期後)」の配当
  • 手順：①配当が一定か成長かを判定　②一定なら D÷r／成長なら 来期配当÷(r−g)
  • 罠：(1)ゴードンの分子は来期配当D₁。当期配当D₀が与えられたら D₁＝D₀×(1+g) に直す。(2)分母は r−g（引き算の順序)。(3)最初の数年だけ高成長の多段階は、高成長期を個別に割り引き、安定期はゴードンでまとめてから現在へ割り引く。

6. サステナブル成長率 〔頻度A〕

解き方の型

  • なぜ：外部から調達せず、内部留保だけを再投資したときに自己資本（と利益）が伸びる率。利益のうち再投資した分（内部留保）がROEで回って増えていく。
  • 式：g＝ROE×内部留保率＝ROE×(1−配当性向)
  • 手順：①内部留保率を出す（＝1−配当性向、または内部留保額÷利益)　②ROEを掛ける
  • 罠：(1)配当性向と内部留保率を取り違えない（足して1になる)。(2)実数（利益・配当・自己資本）で与えられたら、内部留保額＝利益−配当 として g＝内部留保額÷自己資本 でも同じ値が出る。

解答・解説（手順をなぞる）

1. 税引後CF

問1：イ　CFは減価償却前。①減価は所与900。②税引前CF×(1−t)＋減価×t＝2,500×0.7＋900×0.3＝1,750＋270＝2,020。

問2：ウ　与えられたのは利益（減価控除後)。②利益×(1−t)＋減価＝600×0.7＋300＝420＋300＝720。税引後利益420で止めると誤り──減価を足し戻す。

問3：イ　①減価＝3,000÷5＝600。②CFは減価償却前なので1,200×0.7＋600×0.3＝840＋180＝1,020。

2. NPV

問4：ア　①CF確定。②毎期バラバラ→各期に複利現価。③−1,000＋600×0.95＋700×0.91＝−1,000＋570＋637＝207。

問5：ア　毎期一定→年金現価係数を一発掛け。−2,000＋600×4.21＝−2,000＋2,526＝526。

問6：イ　本体（毎期一定）＝1,300×4.33＝5,629。残存売却は最終年の単発なので複利現価で割引＝500×0.78＝390。−5,000＋5,629＋390＝1,019。残存を足し忘れると629（ア)。

問7：ア　CFが毎期違う→個別に複利現価。−2,000＋1,200×0.93＋900×0.86＝−2,000＋1,116＋774＝−110。マイナスもありうる。

3. 投資判定

問8：イ　永続CFのIRR＝CF÷初期投資＝120÷1,200＝10%。10%＞8%→採択。

問9：イ　PI＝現在価値合計÷初期投資＝4,800÷4,000＝1.2。1.2＞1→採択。

問10：イ　A：NPV＝3,900−3,000＝900、PI＝3,900÷3,000＝1.30。B：NPV＝6,200−5,000＝1,200、PI＝6,200÷5,000＝1.24。NPVが大きいのはB、PIが大きいのはA。相互排他なので食い違う＝典型論点。

問11：エ　IRR＝70÷1,000＝7%＜10%→棄却。NPV＝70÷0.10−1,000＝700−1,000＝−300＜0→棄却。両方棄却。

4. WACC

問12：イ　①ウェイトは3,000:7,000＝0.3:0.7。②0.7×10＋0.3×4×0.7＝7＋0.84＝7.84%。

問13：イ　①D/E＝1.0は「1:1」→ウェイト0.5・0.5（1.0ではない)。②0.5×14＋0.5×5×0.7＝7＋1.75＝8.75%。

問14：イ　①ウェイト0.4:0.6。②0.6×10＋0.4×5×0.6＝6＋1.2＝7.2%。(1−t)を忘れると0.6×10＋0.4×5＝8.0（ウ）になる罠。

5. 配当割引モデル

問15：イ　定額型。D÷r＝6÷0.10＝60円。

問16：イ　定率型でD₁が所与。D₁÷(r−g)＝4÷(0.08−0.02)＝4÷0.06＝66.67円。

問17：イ　定率型だが当期配当D₀が所与。まずD₁＝5×1.03＝5.15。5.15÷(0.08−0.03)＝5.15÷0.05＝103円。D₀のまま5÷0.05＝100（ア）とすると誤り。

問18：イ　多段階。D₁＝30、D₂＝30×1.10＝33。第2期末以降は2%成長なので、第1期末時点のゴードン価値＝D₂÷(r−g)＝33÷(0.10−0.02)＝33÷0.08＝412.5（D₂以降をすべて含む)。これを現在へ：P₀＝(D₁＋412.5)÷1.10＝(30＋412.5)÷1.10＝442.5÷1.10≒402円。割引を忘れると442（ウ)。

6. サステナブル成長率

問19：イ　①内部留保率＝1−0.3＝0.7。②ROE×内部留保率＝10%×0.7＝7%。

問20：イ　実数型。内部留保額＝300−120＝180。内部留保額÷自己資本＝180÷2,000＝9%。（検算：ROE＝300÷2,000＝15%、内部留保率＝180÷300＝0.6、15%×0.6＝9%)

問21：ア　内部留保率が直接40%と所与。ROE×内部留保率＝8%×0.4＝3.2%。40%を配当性向と勘違いして8%×0.6＝4.8%（イ）とするのが罠。

使い方（定着のコツ）

  1. 先に「解き方の型」だけ読む → 手順を声に出して言えるか確認。
  2. 問題を解くとき、型の手順番号を指でなぞりながら進める（①ウェイト→②加重平均、のように)。
  3. 間違えたら答えでなく「どの手順で外したか」を特定する（計算ミスか、罠にかかったか)。
  4. 2周目はL2だけを解く（罠への耐性が本番の得点差)。
  5. 数字を変えても手が勝手に動くようになれば、この論点は「身に付いた」状態。

バッチ7：原価計算（L1–L2／回答方法の定着版・19問）

各論点は 解き方の型（なぜ→手順→罠) → L1（素直) → L2（罠込み) の順。
解説は毎回その手順をなぞります。各問4点。標準原価はシュラッター図を使わない基本差異のみ。

1. 総合原価計算（box図の基本) 〔頻度A〕

解き方の型

  • なぜ：大量生産では製品1個ずつ原価を追えない。だから「当月かかった原価」を完成品と月末仕掛品に按分する。材料費と加工費は投入タイミングが違うので別々に按分する。
  • 手順（box図)：
    • ①材料費（始点投入)は数量で按分（月末仕掛品にも全量の材料が入っている)
    • ②加工費は完成品換算量（進捗度×数量)で按分（月末は進捗度の分しか加工されていない)
    • ③按分単価を出す
      • 平均法：単価＝（月初原価＋当月投入)÷(完成品数量＋月末換算量)　←月初と当月を混ぜる
      • 先入先出法：単価＝当月投入原価÷(完成品−月初換算＋月末換算)　←当月分だけ
    • ④月末仕掛品原価＝材料費単価×月末数量＋加工費単価×月末換算量
    • ⑤完成品原価＝月初原価＋当月投入−月末仕掛品原価（差引で出すのが速い)
  • 罠：(1)材料費は数量、加工費は換算量（進捗度を掛ける)。(2)平均法は月初を混ぜ、先入先出は当月分だけで単価を出す。(3)完成品は差引。

2. 総合原価計算（仕損・減損／度外視法) 〔頻度A〕

解き方の型

  • なぜ：製造中に仕損（不良品)・減損（目減り）が出る。正常な範囲なら良品の原価に含める。度外視法は仕損・減損を「最初からなかったもの」として無視し、自動的に良品に負担させる簡便法。
  • 手順：度外視法は仕損・減損の数量を分母から除いて按分するだけ。負担先は発生点で決まる。
    • 終点で発生→完成品のみ負担（月末仕掛品はそこを通過していない)
    • 始点・途中（月末の進捗度より前）で発生→完成品と月末仕掛品の両方が負担
  • 罠：(1)発生点と月末仕掛品の進捗度の大小で負担先が変わる。(2)終点発生なら月末は仕損を負担しない（分母から除くと完成品が自動的に吸収する)。

3. 標準原価計算（直接材料費差異) 〔頻度A〕

解き方の型

  • なぜ：実際原価が出てから「高かった」と言っても遅い。あらかじめ標準（あるべき原価）を決め、実際との差を「価格のせいか、量のせいか」に分けて原因を突き止める。
  • 手順：
    • 価格差異＝（標準価格−実際価格)×実際数量
    • 数量差異＝（標準数量−実際数量)×標準価格
    • 標準数量＝完成品数量×製品1個あたり標準消費量
  • 罠：(1)価格差異は実際"数量"、数量差異は標準"価格"を掛ける（掛ける相手が入れ替わりやすい)。(2)標準＞実際＝有利（コスト安)、標準＜実際＝不利。

4. 標準原価計算（直接労務費差異) 〔頻度A〕

解き方の型

  • なぜ：労務費も「賃率のせいか、時間のせいか」に分ける。考え方は材料費と同じ（価格↔賃率、数量↔時間)。
  • 手順：
    • 賃率差異＝（標準賃率−実際賃率)×実際作業時間
    • 作業時間差異＝（標準作業時間−実際作業時間)×標準賃率
    • 標準作業時間＝完成品数量×製品1個あたり標準作業時間
  • 罠：材料費と同じく、賃率差異は実際"時間"、時間差異は標準"賃率"を掛ける。

5. 個別原価計算（製造間接費の配賦) 〔頻度B〕

解き方の型

  • なぜ：製造間接費（複数製品に共通の費用）は特定の製品に直接跡づけられない。だから配賦基準（直接作業時間など)で各製造指図書に按分する。
  • 手順：
    • 配賦率＝製造間接費総額÷配賦基準総量
    • 各指図書への配賦額＝配賦率×その指図書の配賦基準量
    • 製造指図書の製造原価＝直接材料費＋直接労務費＋製造間接費配賦額
    • 予定配賦差異＝予定配賦額−実際発生額（予定＜実際なら不利)
  • 罠：直接費は跡づけ（配賦しない)、間接費だけ配賦する。配賦基準を取り違えない。

6. 費目別計算（消費価格差異・予定賃率) 〔頻度B〕

解き方の型

  • なぜ：材料の実際価格は仕入のたびに変動し月末まで確定しない。だから予定消費価格で都度計算し、後で実際との差（消費価格差異）を出す。労務費も予定賃率で同様。
  • 手順：
    • 材料消費価格差異＝（予定消費価格−実際消費価格)×実際消費数量（＝予定価格×実際数量−実際消費額)
    • 直接労務費（予定)＝予定賃率×直接作業時間
  • 罠：(1)消費価格差異は実際"消費数量"を掛ける。(2)実際消費額は払出単価の方法（総平均など)で変わる。(3)直接労務費は直接作業時間のみ（間接作業・手待時間は含めない)。

使い方（総合原価のコツ)

  1. まずbox図を描く：左に月初・当月投入、右に完成品・月末仕掛品。材料費と加工費は別の箱。
